# Voltaris Battery Lab

Voltaris — нативное приложение для Windows 10/11, которое показывает телеметрию аккумулятора каждую секунду и измеряет реальную ёмкость в **мА·ч** во время зарядки.

## Возможности

- заряд, статус сети, напряжение, ток, мощность, остаточная энергия и доступная температура;
- заводская и последняя полная ёмкость из отчёта Windows;
- измерение от низкого заряда до 99–100% или частичный тест с экстраполяцией;
- расчёт ёмкости интегрированием тока по времени, а не только чтением паспортного значения;
- 7‑секундный интерактивный инструктаж перед тестом;
- работа в фоне через системный трей;
- OEM-счётчик циклов и собственный счётчик эквивалентных циклов;
- локальная история результатов и экспорт CSV.

## Как считается ёмкость

Windows сообщает мощность зарядки батареи в мВт и напряжение в мВ. Voltaris вычисляет ток `I = P / U`, каждую секунду интегрирует его по времени и получает фактически принятые мА·ч. Для частичного теста результат нормируется на пройденную разницу процентов заряда.

Лучший сценарий — начать с 10–20%, подключить штатный адаптер и зарядить до 99–100% без сна, игр и смены лимита зарядки. Старт с 1% не обязателен и создаёт лишний глубокий разряд.

> Доступность температуры, циклов и части характеристик зависит от ACPI-прошивки ноутбука. Отсутствующие датчики приложение помечает явно.

## Сборка

Требуются .NET 8 SDK и Inno Setup 7 (также поддерживается версия 6).

```bat
build.cmd
```

Результаты:

- `artifacts/publish/Voltaris.exe` — автономное приложение x64;
- `artifacts/installer/Voltaris-Setup-1.0.0.exe` — пользовательский установщик.

Для сборки без установщика:

```bat
build.cmd -SkipInstaller
```

## Хранение данных

История хранится только локально в `%LOCALAPPDATA%\Voltaris\state.json`. Voltaris не отправляет телеметрию в интернет и не управляет зарядкой или режимами питания.
